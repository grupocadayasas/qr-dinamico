ACTUALIZACIÓN: CENTRADO INTELIGENTE DE FOTOGRAFÍAS

Esta actualización analiza automáticamente cada foto al seleccionarla:
- detecta el rostro principal;
- lo centra con margen para cabello, cuello y hombros;
- genera una imagen cuadrada de 1000 x 1000 px;
- comprime la imagen para mejorar la carga;
- si no detecta rostro, aplica un centrado automático seguro.

ARCHIVOS PARA SUBIR A GITHUB
Reemplaza estos archivos en la raíz del repositorio qr-dinamico:
- index.html
- js/admin.js
- assets/styles.css

No modifica js/config.js, Supabase ni los contactos existentes.
No es necesario ejecutar nuevamente el archivo SQL de Storage si la carga de fotos ya funciona.

Después del Commit, espera 1 a 3 minutos y actualiza con Ctrl + F5.
