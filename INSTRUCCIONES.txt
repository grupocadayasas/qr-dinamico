ACTUALIZACIÓN: SUBIR FOTOS DESDE EL PC

1. En Supabase abre SQL Editor > New query.
2. Copia y ejecuta todo el archivo supabase-storage-fotos.sql.
3. En GitHub, repositorio qr-dinamico, sube el contenido de esta carpeta.
4. Acepta reemplazar:
   - index.html
   - js/admin.js
   - assets/styles.css
5. No se modifica js/config.js.
6. Espera el despliegue de GitHub Pages y actualiza con Ctrl + F5.

Uso:
- Entra al generador.
- Abre o crea un contacto.
- Pulsa “Seleccionar foto desde el PC”.
- Elige JPG, PNG o WebP de máximo 5 MB.
- Espera el mensaje “Foto cargada”.
- Pulsa “Guardar contacto”.
